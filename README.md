# 🛒 Walmart Sales Performance Dashboard

## 📌 Project Overview
This project is an interactive Power BI dashboard built to analyze Walmart sales data. It provides insights into sales, profit, orders, product categories, and state-wise performance to support better business decisions.

## 🎯 Objectives
- Analyze overall sales and profit performance.
- Identify top-performing products and categories.
- Track sales trends over time.
- Compare sales across different states.
- Evaluate product-wise profitability.

## 🛠 Tools Used
- Power BI
- Microsoft Excel
- Data Visualization
- DAX
- Power Query

## 📊 Dashboard Features
- Total Sales, Profit, Quantity, and Orders KPIs
- Sales Trend Analysis
- Top 10 Products by Sales
- Profit by Product Category
- Sales by State
- Product Category Analysis
- Interactive Filters (Year, Product Category, State)

## 📈 Key Insights
- Identified top-selling products.
- Compared profit across product categories.
- Analyzed sales performance over time.
- Visualized state-wise sales distribution.
- Improved business insights through interactive dashboards.
